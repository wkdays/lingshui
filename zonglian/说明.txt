启动步骤
1. 安装MySQL，创建数据库：CREATE DATABASE zonglian;
2. 修改 server/db.js 中的数据库连接配置（host、user、password）
3. 进入 server 目录，执行 npm install
4. 执行 npm run init-db 初始化数据库表和测试数据
5. 执行 npm start 启动后端服务（端口8080）
6. 进入项目根目录，执行 npm install
7. 用HBuilderX打开项目，运行到微信小程序
8. 管理后台地址：http://localhost:8080/admin.html
